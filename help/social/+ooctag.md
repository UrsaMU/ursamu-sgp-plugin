+OOCTAG

Set or view your personal OOC tag — the full tag string (decoration
and MUSH color codes included) used in your OOC messages. Overrides
the admin's theme tag for you only.

SYNTAX
  +ooctag                  Show current effective tag + source.
  +ooctag <literal>        Set a personal tag literal.
  +ooctag reset            Clear the override (also: +ooctag clear).

EXAMPLES
  +ooctag [%cyOOC%cn]      Bright-yellow [OOC].
  +ooctag <<%crOOC%cn>>    Double-bracket variant.
  +ooctag (( %cgOOC%cn ))  Parens style.
  +ooctag reset            Back to the game's default.
