@MAIL

In-game mail with drafts, CC/BCC, saved messages, and status flags
[URFS----] (Unread, Replied, Forwarded, Saved). Staff name colors
render on sender/recipient lines.

SYNTAX
  @mail                          List your inbox.
  @mail <#>                      Read message #.
  @mail <player>=<subject>       Start a draft with subject.
  @mail/send                     Send the current draft.
  @mail/proof                    Preview the current draft.
  @mail/abort                    Discard the current draft.
  @mail/subject <text>           Set the draft subject.
  @mail/cc <player>              Add a CC recipient.
  @mail/bcc <player>             Add a BCC recipient.
  @mail/reply <#>                Reply to message #.
  @mail/replyall <#>             Reply to all recipients.
  @mail/forward <#>=<player>     Forward message # to player.
  @mail/save <#> | /unsave <#>   Lock/unlock a message from delete.
  @mail/delete <#>               Delete message # (must be unsaved).
  -<text>                        Add a line to the current draft.
